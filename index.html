One-time setup: OAuth Client ID

This app talks to YouTube directly from your browser using your own Google OAuth 2.0 Client ID. Nothing is sent to any server we control, and no client secret or API key is needed.

  1. Open the Google Cloud Console → Credentials and create (or select) a project.
  2. Enable the YouTube Data API v3 under APIs & Services → Library.
  3. Create credentials → OAuth client ID → application type Web application.
  4. Under Authorized JavaScript origins, add this exact origin: http://localhost:5173
  5. Add yourself as a Test user on the OAuth consent screen (or publish it), then copy the generated Client ID and paste it below.
